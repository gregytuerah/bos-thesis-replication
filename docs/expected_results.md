# Expected Results

The following estimates correspond to the tables in the manuscript and provide
benchmarks for a successful reproduction.

## Primary Outcome: Senior-High Attainment

Preferred model: potential BOS-exposed years interacted with the binary
high-intensity indicator, origin-household fixed effects, female control, and
standard errors clustered by origin household.

| Term | Estimate | Standard error |
| --- | ---: | ---: |
| Exposure (BOS years) | 0.003 | 0.004 |
| Exposure (BOS years) x High Intensity | 0.010* | 0.006 |
| Female | 0.044** | 0.019 |

`N = 2,088`; lower-exposure cohort mean of outcome `= 0.618`.
